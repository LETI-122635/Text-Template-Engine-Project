Purchase Summary
----------------

<rc>
total = 0
count = 0
for item in user.purchases {
    total = total + item.price
    count = count + 1
}
<cr>

Number of purchases: <rc> count <cr>
Total spent: <rc> total <cr>

Details:
<rc>
for item in user.purchases {
    print "- "
    print item.name
    print " : "
    print item.price
    print "\n"
}
<cr>

Analysis:
<rc>
if count == 0 {
    print "No purchases recorded\n"
} else {
    media = total / count
    print "Average per purchase: "
    print media
    print "\n"

    if media > 50 {
        print "High value customer\n"
    } else {
        print "Regular customer\n"
    }
}
<cr>

End of report.
